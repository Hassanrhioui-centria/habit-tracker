# 🧠 Habit Tracker Web App

A simple and intuitive habit tracking application built with React, Node.js, Express, and Mongodb. Users can create, edit, delete, and complete daily habits while tracking their progress and streaks.

## 🚀 Features

- ✅ Add, update, and delete habits (CRUD)
- 🔁 Mark habits as completed for the day
- 🔥 Streak tracking (e.g., 3-day, 7-day streaks)
- 🏆 Achievement badges for consistency
- 🎨 Responsive and modern UI (React + Tailwind CSS)
- 📁 backend using Express + Mongodb

## 🛠️ Tech Stack

- **Frontend**: React, JavaScript, TypScript Tailwind CSS
- **Backend**: Node.js, Express
- **Database**: Mongodb
